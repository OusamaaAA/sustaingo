SustainGo is a mobile application built with Flutter and Django that connects restaurants and NGOs to reduce food waste through a mystery bag model. Users can reserve surprise meals, vendors can upload surplus food, and NGOs can participate in distribution.

## Features
- Role-based dashboards for users, NGOs, and vendors
- JWT authentication and secure sign-in
- Google Maps integration for location-based food filtering
- AI-powered recipe generator using semantic similarity (see related repo)

## Tech Stack
- Frontend: Flutter
- Backend: Django + PostgreSQL
- Authentication: JWT
- Deployment: Render
